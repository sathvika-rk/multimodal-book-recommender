
# 📚 Shelf Picker AI

An interactive, multimodal web application that acts as an instant personal librarian. Users can upload a photo of a physical bookshelf, input their specific reading interest, and receive real-time, tailored recommendations restricted exclusively to the books visible on that shelf.

## 🛠️ Tech Stack
- **Frontend Framework:** Streamlit
- **AI Engine:** Google Gemini 2.5 Flash (`google-genai` SDK)
- **Image Processing:** Pillow (PIL)
- **Language:** Python 3
